# Hello World

This file contains hello world code examples in various programming languages.

## Python

```python
print("Hello, World!")
```

## JavaScript

```javascript
console.log("Hello, World!");
```

## C

```c
#include <stdio.h>

int main() {
    printf("Hello, World!\n");
    return 0;
}
```

```java

        System.out.println("Hello, World!");
    }
}
```
